SamsukaProxy

Локальный SOCKS5-прокси для ускорения Telegram Desktop через WebSocket

Download
⚡

Ускорение

Быстрая загрузка файлов, сообщений и медиа через WebSocket

🔒

Локальный

Работает на 127.0.0.1:1080, полный контроль над трафиком

🔄

Авто-fallback

При недоступности WS переключается на прямое TCP

Как это работает

Telegram → SOCKS5 (127.0.0.1:1080) → SamsukaProxy → WSS (kws*.web.telegram.org) → Telegram DC
  • Поднимает локальный SOCKS5 на 127.0.0.1:1080
  • Перехватывает подключения к IP Telegram
  • Извлекает DC ID из MTProto пакета
  • Устанавливает WebSocket (TLS) соединение
  • Авто-переключение на TCP при недоступности WS
🛡️

Реакция антивирусов

Windows Defender может ошибочно распознать программу как вирус. Это связано с особенностями работы с сетью (перехват трафика) и упаковкой бинарника.

✓ Отключите антивирус на время установки ✓ Добавьте в исключения
